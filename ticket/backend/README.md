# Backend Tickets

Planning tickets for EmX backend work. Each epic lives in its own folder with Jira-style ticket files.

## Epics

| Epic | Folder | Status |
|------|--------|--------|
| Complete Auth System | [auth/](./auth/) | Planned — [30 Jira tickets](./auth/jira/INDEX.md) |

## Ticket ID format

`EMX-AUTH-###` — Auth system tickets  
Future epics: `EMX-<DOMAIN>-###`

## How to use

1. Read the epic overview in the epic folder.
2. Work tickets in dependency order (see epic README).
3. Move ticket status from `To Do` → `In Progress` → `Done` in each file header when executing work.
